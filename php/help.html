Πώς να Χρησιμοποιήσετε την Πλατφόρμα

Η πλατφόρμα παρέχει διάφορα εργαλεία για τη διαχείριση των λιστών εργασιών σας. Εδώ θα βρείτε μερικές βασικές οδηγίες για να ξεκινήσετε:

  • Για να δημιουργήσετε μια νέα λίστα εργασιών, μεταβείτε στον πίνακα ελέγχου και κάντε κλικ στο "Δημιουργία Νέας Λίστας".
  • Για να προσθέσετε μια νέα εργασία σε μια λίστα, κάντε κλικ στο "Προσθήκη Εργασίας" δίπλα στη λίστα στην οποία θέλετε να προσθέσετε την εργασία.
  • Για να επεξεργαστείτε ή να διαγράψετε μια εργασία, κάντε κλικ στα αντίστοιχα κουμπιά δίπλα στην εργασία.

Συχνές Ερωτήσεις

Πώς μπορώ να αλλάξω το προφίλ μου;

Μπορείτε να αλλάξετε το προφίλ σας κάνοντας κλικ στο "Επεξεργασία Προφίλ" στον πίνακα ελέγχου και ενημερώνοντας τις πληροφορίες σας.

Πώς μπορώ να αλλάξω το θέμα της πλατφόρμας;

Μπορείτε να αλλάξετε το θέμα της πλατφόρμας κάνοντας κλικ στο κουμπί "Αλλαγή Θέματος" στην κορυφή της σελίδας.
